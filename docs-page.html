Installation

UiPath Assistant


Step 1: Make an account on Orchestrator and get an enterprise account for the orchestrator.
Step 2: Get a link on your designated email ID for logging into the cloud.UiPath.
Step 3: On the home tab, there will be a clickable link on the bottom right side named “Resource Center”. Under the heading featured downloaded ,in the section assistant for Windows click on the link which suits your system version.
For example, for Windows it would be Windows XP,Windows 10 and Windows 7




Microsoft SQL Server and SQL Server Management Studio


Step 1: Click on SQL Server and download SQL Server 2019 Express.
Step 2: After installing it ,on the final screen a button to install SSMS will appear (SQL Server Management Studio). Install the studio through that on the system.
(Note: Make sure the version of SQL Server is the same as that of Management studio)

Intergrations

UiPath Assistant



Step 1: Open the UiPath Assistant from the system and click on sign in button. Enter the details (username/password) which were fed into the orchestrator.
Step 2: Click on the name in the top right corner and select preferences. Go to orchestrator settings.
Step 3: Choose Machine key as the connection type and ask for your machine name from IT department.
Step 4: Go to the orchestrator and copy the url till before “/orchestrator_” .For example, the url should look something like this https://cloud.uipath.com/jatayu/MailTrigger. Here, “jatayu is the organisation's name and “MailTrigger” is the tenant name mentioned in the upper right corner of the orchestrator dashboard.
Step 5: Now for the machine key, go on to the orchestrator dashboard and click on tenant block situated just beside the leftside nav bar and select “Machines”.
Step 6: Copy the Client ID as shown in the screenshot below and paste it on the UiPath assistant.



Step 7: Click on Connect button and Voila! A green light should appear on the top right corner and it is connected to the orchestrator.
(Incase of license unauthorised error,please check that you have chosen the organisation's tenant and not default tenant)

One Drive(Power Automate Integration)


Login to One drive through the business ID provided which will initiate the connection with the Power Automate. Hence triggering the alerts automation.
This connection is completely automated,hence the user doesn't need to worry about triggering any button for the process.

Web



A splash screen appears which will take the user to the home page and it will display the slider where the inspiring words “Imagine.Invent.Inspire” and an exuberating statement such as “Feel the RPA Touch”.


Sliding down,the services provided by the team blaze will be visible. By clicking on each service,a more detailed description of those services will be displayed on a pop-out screen.The services include Reports &Alerts ,Automation,Seamless Communication,Progress Tracking and Secure. An AboutUs Section and profiles of all the developers can be seen by further scrolling down and the OnBoarding phases which will give a brief about the process can be found.The footer will include the quick links and contact links to our developer team. There will be a login button in the header which will redirect the user to the login page.


On the Login page , the user can feed in the login details given in the mail by the organisation which are username and password and then filling them cautiously will take the user to Enable to Onboarding screen otherwise an error will pop up such as “Invalid credentials”.


On the next screen a button named”Enable to Onboard” can be seen at the centre of the screen and a side navbar on the right hand sidecan be seen with a bot icon at the top and consecutively other icons for different pages . Clicking on the “Enable to Onboard” button will start the mail triggering process at the backend ,sending the necessary emails to all the different departments


This is to ask them for updates about new the employee's onboarding process ,where they will either confirm or tell its on hold updating the HR Progress Report ,which can be seen by clicking the Progress Report icon in the side nav bar .


A home button is given to take the user to the home screen. Also, the Modify Table icon button takes the user to Modify Employee Data screen where a table with the headings First Name, Last Name ,Joining Date and MailID is displayed under which the HR can fill in the details of the New Employee. Other options such as add,edit,delete,update and save are also available in the right most column of the table. Entering any value out of specified constraints will display an error message”Invalid Input”.For example , entering any special characters in the first name,last name or entering jibberish words in MailID will result in an error


Furthermore, the Alerts page will display the timeline and the body of alerts that are being delivered to the user and an interactive piechart is displayed for the user to understand the data. Also clicking on the downward arrow icon beside the column heading such as body ,the user can sort,filter or search through names and date in order to filter out any important alerts.


Lastly, clicking on the Outlook icon in the side navbar will redirect to the dashboard of the outlook screen.and the Logout button will redirect the user to the login page. Bravo! You have a basic understanding of how to operate the website.
